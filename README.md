# 프로젝트명

TIMWORK

## 실행 방법

```
git clone https://github.com/nas7062/timwork.git .
npm install
npm run dev
```

## 기술 스택

### Core
```
React 19
TypeScript 5
Vite 7
```
### Routing & State
```
React Router v7
Zustand
```
### Styling
```
Tailwind CSS v4
clsx
```

## 구현 기능

| 기능          | 설명                                                                                         |
| ------------- | -------------------------------------------------------------------------------------------- |
| 도면 탐색     | 공간, 공종, 영역, 리비전을 선택해 사용자가 원하는 도면을 찾을 수 있음                        |
| 도면 표시     | 선택한 도면 이미지를 화면에 표시                                                             |
| 컨텍스트 인식 | TopBar를 통해 현재 어떤 도면을 보고 있는지 사용자가 알 수 있도록 하였음                      |
| 오버레이      | 오버레이를 통해 공종, 리비전간에 도면을 겹쳐서 충돌하는지 혹은 무엇이 바뀐지 확인할 수 있음. |
